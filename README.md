- Course Management

## 🛠 Tech Stack

- React
- TypeScript
- Vite
- Node.js

---

## 📥 Installation

Clone the repository

git clone https://github.com/sakshipawar5840/edumanage.git

Go to project folder

cd edumanage

Install dependencies

npm install

Run the project

npm run dev
